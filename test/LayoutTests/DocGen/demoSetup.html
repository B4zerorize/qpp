Querypoint Demo Setup

  1. Install the Chrome 'dev channel' release (26.0.1398 or newer)
  2. Open chrome://flags and set experimental extension APIs enable, then relaunch the browser
  3. Install the Querypoint Debugger
  4. Open the demo page http://localhost:8686/test/qpd.html
  5. Right Click on the demo page and select DevtoolsExtended -> Debug With DevtoolsExtended
  6. Begin the demo
