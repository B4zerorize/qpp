The Querypoint Panel, First Look

Click the Querypoint Tab to Open the Querypoint Panel. The first time you open the panel, a file selection dialog comes up. To select files later, use control + o

Select demo.js by clicking on it, typing it in to the filter, or using arrow keys plus enter.

Viewing source file demo.js

The source for demo.js appears on the right hand side; mouse over the source to see the left-hand-side token information change.
